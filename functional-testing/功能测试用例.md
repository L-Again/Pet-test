# 功能测试用例

## 用例字段

| 字段 | 说明 |
| --- | --- |
| 用例编号 | `FUNC_模块_编号` |
| 模块 | 页面或业务模块 |
| 测试标题 | 具体功能点 |
| 优先级 | P0/P1/P2 |
| 前置条件 | 登录状态、角色、测试数据 |
| 测试步骤 | 页面操作步骤 |
| 测试数据 | 输入数据 |
| 预期结果 | 页面展示、提示、数据状态 |
| 实际结果 | 执行后填写 |
| 是否通过 | 通过/失败 |
