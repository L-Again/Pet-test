# 接口测试用例

## 用例字段

| 字段 | 说明 |
| --- | --- |
| 用例编号 | `API_模块_编号` |
| 模块 | 接口所属模块 |
| 用例标题 | 具体测试点 |
| 优先级 | P0 核心主流程，P1 重要异常，P2 一般异常 |
| 前置条件 | Token、已有数据、角色 |
| 请求方法 | GET/POST/PUT/DELETE |
| URL | 接口路径 |
| 请求头 | Content-Type、Authorization |
| 请求数据 | JSON、form-data 或 path 参数 |
| 预期结果 | HTTP 状态、业务 code、数据变化 |
| 实际结果 | 执行后填写 |
| 是否通过 | 通过/失败 |
