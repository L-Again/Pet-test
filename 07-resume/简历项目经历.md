# 简历项目经历

## 项目名称

流浪动物救助与领养平台测试项目

## 项目描述

该项目为前后端分离的流浪动物救助与领养平台，包含用户注册登录、救助信息发布与审核、救助评论、管理员收编、领养宠物管理、领养申请、沟通消息、用户签收、角色权限控制等功能。本人基于本地部署环境，围绕核心业务流程完成测试计划设计、功能测试、接口测试、权限测试、基础性能测试和接口自动化回归。

## 项目职责

1. 梳理普通用户和管理员两类角色的核心业务流程，设计功能测试用例，覆盖注册登录、救助发布、审核、评论、收编、领养申请、沟通、签收等场景。
2. 使用 Postman 进行接口测试，配置环境变量和 JWT Token 鉴权，覆盖正常请求、异常参数、未登录、越权访问、重复提交、错误状态流转等测试点。
3. 基于 MySQL 验证关键业务数据落库和状态流转，包括救助状态、领养申请状态、宠物状态、删除标记和用户角色。
4. 使用 Shell + curl 编写接口自动化回归脚本，覆盖救助和领养核心主流程及驳回分支，提高回归验证效率。
5. 使用 JMeter 对健康检查、登录、救助列表、领养宠物列表等核心接口进行基础并发测试，关注平均响应时间、P95、吞吐量和错误率。
6. 整理测试用例、接口测试说明、回归脚本说明、性能测试方案、缺陷记录和项目面试话术，形成 GitHub 测试作品集。
